❯ Создай кроссплатформенное (macOS + Windows) портативное desktop-приложение    
  для генерации и вшивания субтитров в видео с использованием Whisper и FFmpeg. 
                                                                                
  ## СТЕК ТЕХНОЛОГИЙ                                                            
  - **UI**: Tauri + React + TypeScript + TailwindCSS (легковесный бандл,        
  нативная производительность, современный стиль)                               
  - **Backend**: Python (whisper / faster-whisper, ffmpeg-python), запускается  
  как sidecar-процесс через Tauri                                               
  - **Архитектура**: UI ↔ локальный API (Tauri commands или localhost) ↔ Python 
   воркер                                                                       
  - **Whisper**: всегда модель `large-v3` (НЕ давать пользователю выбор модели) 
  - **GPU-ускорение**: Metal (MPS) на Mac, CUDA на Windows если доступно, иначе 
   CPU fallback                                                                 
                                                                                
  ## ПОРТАТИВНОСТЬ (КРИТИЧНО)                                                   
  - Все настройки, кеши, модели хранятся в папке РЯДОМ с исполняемым файлом, НЕ 
   в системных папках пользователя                                              
  - Структура: `./app.exe` (или `.app`) + `./data/settings.json` +              
  `./data/models/` + `./data/ffmpeg/` + `./data/logs/`                          
  - Отдельная вкладка "Setup" / "Установка": при первом запуске или по кнопке   
  скачивает Whisper large-v3 и FFmpeg в локальную папку, прописывает пути       
  автоматически                                                                 
  - При переносе папки на другой Mac/Windows — всё работает без переустановки   
                                                                                
  ## ДИЗАЙН                                                                     
  - **Палитра**: чёрный фон (#0a0a0a — #1a1a1a), золотые акценты (#d4af37,      
  #f4d03f), белый текст                                                         
  - **Эстетика**: тёмное матовое стекло (frosted glass / backdrop-blur), мягкие 
   золотые radial-gradient засветы на фоне (приглушённые, не отвлекают)         
  - **Анимации**: плавные transitions (200-300ms), hover-эффекты с золотым      
  свечением, золотая "вспышка" при завершении транскрипции                      
  - **Иконки**: Lucide React, золотой цвет                                      
  - **Шрифт UI**: Inter или SF Pro                                              
  - **Drag & drop**: видео можно перетаскивать прямо в окно                     
  - Минимализм: интерфейс не перегружен, золотые элементы используются          
  акцентно, не доминируют                                                       
  - Запоминание всех настроек между запусками (в settings.json)                 
                                                                                
  ## ВКЛАДКИ ПРИЛОЖЕНИЯ                                                         
  1. **Main** — основной экран обработки                                        
  2. **Style** — настройки стиля субтитров + превью                             
  3. **Queue** — очередь файлов / пакетная обработка                            
  4. **Setup** — скачивание моделей Whisper и FFmpeg, проверка путей            
  5. **Settings** — общие настройки (горячие клавиши, уведомления, тема)        
                                                                                
  ## ФУНКЦИОНАЛ ТРАНСКРИПЦИИ                                                    
  - Выбор языка транскрипции (выпадающее меню со ВСЕМИ языками которые          
  поддерживает Whisper, ~99 языков, с человекочитаемыми названиями)             
  - Опция "Auto-detect language"                                                
  - VAD (Voice Activity Detection) для очистки тишины — включено по умолчанию   
  - Word-level timestamps (обязательно — нужно для правильной разбивки строк)   
  - Опциональная очистка от слов-паразитов (эээ, ммм, uh, um) — чекбокс         
  - Опциональный перевод на английский (Whisper translate mode) — чекбокс       
                                                                                
  ## ПРАВИЛА ФОРМИРОВАНИЯ СУБТИТРОВ (КРИТИЧНО)                                  
  - Настройка "Максимум символов в строке" (слайдер или input, например 20-80)  
  - Субтитры ВСЕГДА в одну строку (никаких многострочных субтитров)             
  - НИКОГДА не разрывать слово посередине — если слово не помещается в текущий  
  лимит, целиком переносится на следующий тайминг                               
  - Все слова должны быть закончены (использовать word-level timestamps Whisper 
   для точной нарезки)                                                          
  - Настройка минимальной длительности субтитра (чтобы не мелькал)              
  - Настройка максимальной длительности субтитра (длинные фразы разбиваются)    
  - Настройка скорости чтения CPS (characters per second) — автоматическое      
  выравнивание таймингов под комфортное чтение                                  
                                                                                
  ## НАСТРОЙКИ СТИЛЯ СУБТИТРОВ                                                  
  - **Шрифт**: выпадающее меню со шрифтами установленными на компьютере         
  пользователя (читать системные шрифты через ОС API)                           
  - **Цвет текста**: color picker                                               
  - **Жирность / курсив**: чекбоксы (Bold / Italic)                             
  - **Размер текста**: слайдер (например 12-100 px)                             
  - **Цвет подложки (фон субтитра)**: color picker                              
  - **Непрозрачность подложки**: слайдер 0-100%                                 
  - **Скругление углов подложки**: слайдер (border-radius)                      
  - **Внутренние отступы подложки** (padding): слайдер                          
  - **Обводка текста (outline)**: цвет + толщина (важно для читаемости)         
  - **Тень текста**: цвет + смещение X/Y + размытие                             
  - **Позиция на экране**: сверху / по центру / снизу + слайдер отступа от края 
                                                                                
  ## ПРЕВЬЮ                                                                     
  - При выборе видео автоматически извлекается первый кадр (через FFmpeg)

  или первая распознанная фраза если уже есть транскрипция) с ТЕКУЩИМИ
